1. Locate where your dataset of images is saved
2. Copy the folder's path into the variable master_path_to_dataset
3. If you wish to start at a certain timestamp, fill in the startTimestamp variable
4. If you wish to see the whole image (uncropped), change the cropDisparity value to False
5. Run the program
6. Controls during runtime:
    exit - x
    save yolo and disparity map - s
    crop - c
    pause the 'video' - space
